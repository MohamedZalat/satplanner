Ce dossier contient le fichier python permettant d'afficher les résultats du planner utilisé dans le projet et le planner ASP.

Pour que les figures s'affichent comme il se doit, quelques consignes doivent être respectés :
-le fichier doit avoir pour séparateur ";"
-le nom des colonnes pour notre planner doit être : problems;valeurs Planner
-le nom des colonnes pour ASO doit être ; problems;valeurs ASP

Pour lancer le programme, deux arguments sont attendus :
les chemins vers les deux fichiers que l'on souhaite afficher.
La figure est automatiquement enregistrée (cependant, il faut modifier manuellement le nom des figures avant enregistrement)
